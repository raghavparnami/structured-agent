2. Agent State Machine

stateDiagram-v2
    [*] --> RETRIEVE

    RETRIEVE --> GENERATE : schema context ready

    GENERATE --> VALIDATE : SQL generated

    VALIDATE --> EXECUTE : valid
    VALIDATE --> REPAIR : invalid

    REPAIR --> VALIDATE : fixed SQL
    REPAIR --> FAILED : max 3 retries

    EXECUTE --> SYNTHESIZE : results
    EXECUTE --> REPAIR : execution error

    SYNTHESIZE --> DONE : answer ready

    DONE --> [*]
    FAILED --> [*]

6. Feedback Learning Loop

flowchart TB
    Q["User asks question"] --> SQL["Agent generates SQL"]
    SQL --> RES["Show results + SQL"]

    RES --> UP["Thumbs Up"]
    RES --> DOWN["Thumbs Down"]
    RES --> CORR["User corrects SQL"]

    UP --> SAVE["Save as verified
few-shot example"]
    DOWN --> LOG["Log for review"]
    CORR --> EXTRACT["LLM extracts:
new business terms
+ default filters"]
    CORR --> SAVE

    SAVE --> DEDUP["Deduplicate
similarity > 0.95 = merge"]
    DEDUP --> PRUNE["Prune
max 100 examples
keep all verified"]
    PRUNE --> DISK["few_shot_examples.json"]

    EXTRACT --> TERMS["learned_business_terms.json"]
    EXTRACT --> GRAPH["Update Knowledge Graph"]

    DISK --> NEXT["Next query:
retrieved as few-shot context"]
    TERMS --> NEXT
    GRAPH --> NEXT
